# AGENTS.md

이 문서는 이 저장소에서 작업하는 모든 에이전트/기여자를 위한 필수 운영 규칙입니다.

## 필수 규칙

1. 코드 또는 동작 변경이 발생하면 반드시 `README.md`를 함께 업데이트한다.
2. `README.md` 업데이트가 없는 기능/버그 수정은 작업 완료로 간주하지 않는다.
3. 커밋 전 아래를 확인한다.
   - 무엇이 바뀌었는지(`기능`, `설정`, `API`, `실행 방법`, `주의사항`)
   - 해당 변경이 `README.md`에 반영되었는지
4. 변경 영향이 사용자 사용법에 닿는 경우(`CLI`, `웹 대시보드`, `환경 변수`, `출력 파일`, `수신자/알림 정책`)는 반드시 README에 반영한다.
5. 문서 업데이트를 의도적으로 제외하는 경우, 커밋 메시지 또는 PR 설명에 제외 사유를 명시한다.

## 권장 절차

1. 기능 구현
2. 동작 검증
3. `README.md` 변경 반영
4. 커밋
